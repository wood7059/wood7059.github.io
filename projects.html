-Threaded Map

In a data structures course we had an assingment to created and organized threaded map that searches at 0(n). The items in each node is a key and some sort of value which could be a character, string, or object. This algorithm is effective at storing data that has a good runtime when creating new data fields, searching for existing data fields, or deleting data fields.
